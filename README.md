# British Airways Data Analytics Job Simulation

## Overview
This project is based on a job simulation experience focused on analyzing customer review data to generate actionable business insights.

## Dataset
The dataset was provided in extracted format as part of the simulation. The focus of this project is on data cleaning, analysis, and insight generation.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook
- Excel
- AI tools for code assistance

## Key Learnings
- Cleaning and preparing real-world datasets
- Exploratory data analysis
- Translating data insights into business recommendations

## Disclaimer
This project does not use real British Airways internal data and is intended solely for learning and demonstration purposes.
